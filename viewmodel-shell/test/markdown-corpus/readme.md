# ExampleLib

`ExampleLib` is a small illustrative library used to exercise the markdown → ViewNode converter.

## Install

```bash
npm install example-lib
```

## Usage

```typescript
import { example } from "example-lib";

example({ answer: 42 });
```

Read the [full documentation](https://example.com/docs) for details, or open an issue on the [tracker](https://example.com/issues).

## Features

- Zero dependencies at runtime
- Ships **TypeScript** types out of the box
- Well-tested — every public API has a corresponding *unit test*
- Supports both ESM and CommonJS
